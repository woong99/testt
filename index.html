창의 개념

창 : 세계 좌표계에서 지정된 사각 부분

시역: 세계 좌표계에서 정의된 창이 출력 장치에서 생성되는 사각 영역

관측변환: 창 내의 화상을 원하는 시역으로 옮기는 과정

창의 경계 정의 : 세계 좌표계에서 지정

시역의 정의 : 출력 장치가 하나이면 그 장치의 좌표계로 시역을 정의, 보편적으로는 정규회된 장치 좌표계로 시역을 정의

  • 정규화된 장치 좌표계의 좌표값은 0에서 1까지의 값을 가짐
  • 장치에 독립적
  • 장치 구동기를 사용하여 각각의 출력 장치들에 출력

시역의 위치나 크기를 바꾸면 대상물의 위치 및 크기와 비율이 바뀜 : 줌 - 창이 작아지면 시역에서는 확대되어 보임, 패닝 - 고정된 크기의 창을 움직일 떄 일어나는 현상

여러 개의 창과 시역을 설정하는 방법 : 각 부분마다 레이블을 붙임, 창과 시역 명령어에 매개 변수를 추가함 - 매개 변수는 정수 색인값으로 할당, 시역이 겹쳐지는 경우 : 각 시역의 레이블에 우선 순위를 부여

여러 개의 워크스테이션에 대한 창과 시역의 구현 : 창과 시역 명령어에 워크스테이션 번호를 부여, 최종적인 화상의 각기 다른 부분들을 각기 다른 출력 장치에 표시

좌표축에 평행하지 않은 다른 형태의 창이나 시역의 정의 방법 : 창 명령에 각을 나타내는 매개 변수를 추가
